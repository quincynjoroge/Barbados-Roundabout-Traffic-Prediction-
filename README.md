This project aims to **predict traffic congestion in a specific roundabout in Barbados** using machine learning, with the goal of identifying the **root causes of congestion**. The insights generated from this analysis can help the Ministry of Transport and Works design interventions to improve traffic flow across the island.  

## Project Overview

On the small island of Barbados, cars, buses, and taxis are the primary modes of transport, and traffic congestion is a significant problem affecting daily life. This project uses **video data streams labeled with congestion ratings** to predict traffic congestion **five minutes into the future**. The ultimate objective is to **understand contributing factors** to traffic delays.
